    Python API Reference¶

    This is the functions reference of MNE-HCP. Functions are grouped thematically by analysis stage. Functions that are not below a module heading are found in the hcp namespace.

    make_mne_anatomy(subject, anatomy_path[, ...]) Extract relevant anatomy and create MNE friendly directory layout
    compute_forward_stack(anatomy_path, subject, ...) Convenience function for conducting standard MNE analyses.

    Reading HCP files¶

    hcp.io:

    read_raw_hcp(subject, data_type[, ...]) Read HCP raw data
    read_epochs_hcp(subject, data_type[, onset, ...]) Read HCP processed data
    read_evokeds_hcp(subject, data_type[, ...]) Read HCP processed data
    read_info_hcp(subject, data_type[, ...]) Read info from unprocessed data
    read_annot_hcp(subject, data_type[, ...]) Read annotations for bad data and ICA.
    read_ica_hcp(subject, data_type[, ...]) Read precomputed independent components from subject
    read_trial_info_hcp(subject, data_type[, ...]) Read information about trials

    hcp.io.file_mapping:

    get_file_paths(subject, data_type, output[, ...]) This is the MNE-HCP file path synthesizer
    get_s3_keys_meg(subject, data_types[, ...]) Helper to prepare AWS downloads for MEG data
    get_s3_keys_anatomy(subject[, ...]) Helper to prepare AWS downloads for anatomy data

    Manipulating data and sensors¶

    hcp.preprocessing:

    set_eog_ecg_channels(raw) Set the HCP ECG and EOG channels
    apply_ica_hcp(raw, ica_mat, exclude) Apply the HCP ICA.
    apply_ref_correction(raw[, decim_fit]) Regress out MEG ref channels
    map_ch_coords_to_mne(inst) Transform sensors to MNE coordinates
    interpolate_missing(inst, subject, ...[, ...]) Interpolate all MEG channels that are missing

    Visualizing data¶

    hcp.viz:

    plot_coregistration(subject, anatomy_path, ...) A diagnostic plot to show the HCP coregistration
    make_hcp_bti_layout(info) Get Layout of HCP Magnes3600WH data
